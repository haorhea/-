                        <div class="txt js-content ptag"><p>花京院高一暑假前，学校组织了一次类似于远足的活动，清晨出发，要在外面过上两晚。花京院在那个即将出发的清晨的八小时前才告诉他这件事，承太郎从电脑屏幕上面抬起头来，难以置信地看着孩子，花京院很执拗地也盯着他。这眼神反而让他无话可说了，最后只好关照了一句：行李理好了吗？</p> 
<p>花京院看他一眼说，你当我几岁了？</p> 
<p>说完这句话之后，高中生就回头躲进房间去睡觉了。海洋学者久久地盯着他离开的方向，最后颇为失落地低头继续面对自己的数据表。他这天晚上的工作其实并没有那么多，数据大致整理整理，第二天派给研究生做个拟合就行，可不知为何突然一股气冲上脑门，硬是把前后一个月的数据都人工查了一遍，自己重新建模，算参数，拟合，预测未来数据，这工作单纯是数学方面的，对他来讲非常乏味，当然并不乐在其中，但做着做着，忘了时间，再抬头的时候又看到了花京院，高中生从门缝里望着他，皱着眉，不赞同地说：你一晚没睡？</p> 
<p>承太郎将眼光从花京院那皱成一团的脸上移开，拉着窗帘的落地窗那边已经透出了蒙蒙的光，他这才发现原来天都亮了。花京院已经换好了衣服，他走到他身边，抱着胳膊，看着他的电脑屏幕，低声说你当你几岁了，这样下去迟早身体要跨，要不就眼睛会瞎。承太郎揉揉眼睛站起身来，有点尴尬地听到自己的膝盖和腰都在久坐不动之后发出卡拉卡拉的响声，他推着花京院离开书房，打着哈欠说，我给你弄早饭吃。</p> 
<p>“不用，我去学校路上买点东西吃就行了。”</p> 
<p>“外面买的不好。”</p> 
<p>“你别管我早饭的事了，”高中生走到半路，回过头抓着他的胳膊，很强硬地把他往回推，“快去睡觉，一会儿我出门的时候帮你关电脑。”</p> 
<p>“我还要上班呢。”</p> 
<p>“离你上班时间还有四个钟头，快点去睡，我帮你定个闹钟。”</p> 
<p>他最后还是被花京院推回了自己的房间，在他的注视下，四肢僵硬，脑袋昏沉地挪上了床，一面躺下去一面还在问他：学生证带了没？钥匙带了没？钱带了多少？够不够？</p> 
<p>得到的回答都非常敷衍，无非就是带了带了你别担心你快点睡觉，一边说还一边帮他把薄被盖好，承太郎放松下来，心想他的确也是个大孩子了，自己或许是有些多嘴，于是渐渐就睡了过去，意识模糊间听到轻轻的关门声，花京院离开了。</p> 
<p>这夹在通宵加班和朝九晚五之间的四个小时睡眠本来在某种意义上还算惬意，他久违地做了个最近半年来都没有做过的梦，梦中的自己和花京院都停留在十七岁，而且就和当年他们十七岁共处的时候一样，是一对年轻而不知餍足的恋人，梦中的他和花京院很自然地拥抱，接吻，很自然地在一张梦中自动出现的大床上做爱，花京院挂在他身上，咬他的耳朵，咬他的锁骨，咬他的胎记，疼痛与麻痒并行的微妙触感一圈圈地在身体内外扩散开来，承太郎看着身下的恋人，花京院不乱咬人的时候喜欢咬着自己的嘴唇，长而纤薄的唇因此扭曲得很厉害。承太郎腾出一只手去拨弄那被主人蹂躏到发白的下唇，循循善诱，引导他说些什么，花京院于是张开了嘴，凑到他耳朵边，清晰而动人地说道：</p> 
<p>“承太郎，我爱你。”</p> 
<p>“我也爱你。”</p> 
<p>他用语言和行动同时热情地回应着恋人的告白，越发深入地开拓着对方主动缠上来的身体，其实他还有很多话想说，想告诉他自己有多喜欢他，想给出承诺，想让对方知道自己此刻有多么愉悦满足，然而说不出来，脑海中混混沌沌的一团，再多信息到了嘴边都只能压缩成一句略显单薄的，我爱你。只是光这一句似乎也已经足够了，花京院在笑，他的四肢缓缓贴上来，声音悦耳清亮：</p> 
<p>承太郎，我爱你。</p> 
<p>永远永远。</p> 
<p>我喜欢你。</p> 
<p>我想永远都和你在一起。</p> 
<p>亲密的爱语渐渐变了味，直到最后一句话，像句恶毒的诅咒，仿佛王子在一夜狂欢之后才猛然发现身边的黑天鹅竟然是魔法师的女儿，而非他所心心念念的奥黛特公主。他停下动作再次低头望去，柔软的被单枕头之间一张再熟悉不过的脸，染着情欲的饥渴和羞涩的绯红色，花京院依旧缠着他，但是这张脸似乎比十七岁要年轻那么一些。冷汗几乎是一股脑地从后背冒出来，承太郎下意识地想少许离远，动作的时候，膝盖腰腿却都卡拉卡拉地响，花京院牵着他的手，笑着说别突然停下来啊，你以为你几岁了。</p> 
<p>我三十四岁了。这个念头撞进脑子，在唇舌边徘徊，又因恐惧而不敢轻易吐露。我三十四岁，花京院十五岁，他是我养大的，他在我床上。</p> 
<p>十五岁的花京院十分温柔地拉着他的手，少年人柔韧而光洁的大腿打开一个更大的角度，无言地邀请他继续刚才的情事。理智几乎是催促着他从那张温柔旖旎的大床上落荒而逃，欲望却没有被负面情绪压灭，他也没有甩脱花京院牵着他的那只手，气喘吁吁，眼红耳热，在本能和自制之间两难。</p> 
<p>好在这纠结并没有维持太久，闹钟响了，梦境随之分崩离析，承太郎在床上睁开眼，鬼压床似的久久不能动弹。</p> 
<p>这天去了研究室之后正好又被教授邀到学校，说是去给本科生们随便讲两句，邀请者是他恩师，态度诚恳又亲热，不好拒绝，于是只好勉强前往。他在车程上试图给演讲打个腹稿，然而无法专心，每一个原本都无比正常，和暧昧丝毫不沾边的词都会奔着不对的方向疾驰而去，水温，软体，繁衍，无脊椎，随便一个术语都在他脑内自动被引向几个小时前梦中的场景，他家那个十五岁孩子苍白而略带粉红的肢体，瘦骨嶙嶙，又柔韧得不可思议。</p> 
<p>海洋学者捂着眼睛，心想哪怕是心思最活络的思春期男生也不至于弄得像是自己现在这样，满脑子都是那些东西吧。耳边传来老师的声音，问他，空条，怎么了，难不成是晕车，不至于吧，你可是经常出海的人啊。他只能答以睡眠不足，并同样用这个借口解释了之后的短演讲中，自己为何信口开河，表现得和往日严谨的作风大相径庭。</p> 
<p>这天过得格外疲累漫长，唯一的慰藉是晚上回到家的时候，他不必立刻面对自己的淫梦对象。独自晚餐洗漱之后他就倒上了床，累是真的，但并不很想睡，脑子里那些东西翻来覆去一整天，身体随时处在紧绷状态，一个人放松独处之后反而有了反应。承太郎倒在床上自暴自弃地解决了一下，他平时很少做这种事，偶尔压力与欲望都积攒到一定极值才会像完成任务一样动手，自慰时的素材当然是花京院，只是往常一直都觉得自己想的当然是那个十七岁，和自己有过恋爱之实也的确发展到了那一步的花京院典明，直到今天才猛然发现，原来那个十余年来一点点养大的孩子早就潜移默化地替换取代，连他那寡淡的性幻想都逐渐入侵了。</p> 
<p>花京院到了暑假里就将十六岁，是个完全可以恋爱，或者说得直白些，完全可以带上床的年纪了。如果承太郎依旧是当年的十七岁，在发现恋爱苗头的一开始就会猛烈进攻，一举拿下，接着将生米煮成熟饭，连犹豫都不会怎么犹豫。然而他毕竟不再是当年那个一往无前的高中生了，空条承太郎三十四岁，十多年都没有谈过恋爱，十多年来都处在监护人的立场上，他所能做的最多是说服自己接受眼下已经发生的事实，至于在接受之后立刻为此付诸行动，那绝不可能。</p> 
<p>他想起那个晚上，花京院躺在他身边说了我爱你，事后又气急败坏地逃掉了，说你什么也不懂。此刻想起来，总觉的读出不少不应该读出来的意思，但自己这边的感情已经变了味，于是立刻又觉得自己是在自作多情。这自作多情于他看来并不病态，毕竟承太郎一直觉得自己必然会爱上花京院，无论哪个，那都是件顺理成章水到渠成的事，只是到了现在的情况中，虽然能自我意识并自我肯定，却不会主动出手去做些什么了。</p> 
<p>毕竟他可不希望花京院的“我不想看到你”，最终因为自己的轻举妄动，变成了“我永远都不想再看到你。”</p> 
<p>&nbsp;</p> 
<p>七月中旬，花京院放了暑假，承太郎研究室里几篇论文刚刚投出去，于是也得到了十五天的短休。他选择待在书房里翻翻闲书，看些和专业无关的文学作品，花京院则每天顶着烈日出门，到了差不多是平时放学的时间再回家来，问他是去干什么了，答曰和同学一起去了图书馆。</p> 
<p>空条承太郎年轻的时候是个醋劲很大的人，如果说当年花京院典明是自尊太强，因此常常生气，那他就是因为嫉妒心太强而天天吃醋，远到什么商店里多嘴了两句的陌生女性，近到波鲁那雷夫，全都遭过他莫名其妙的嫉妒。领养的这个花京院上了学有了朋友之后，他站在家长的立场上，尽量表现得开明又善解人意，于是这醋劲就很多年没有发作过，然而十年过去，此刻听到对方说是跟朋友出去又耗了一天的时候，那熟悉的酸溜溜的感觉竟然又从心里某个角落悄悄抬起了头。承太郎边往嘴里送着白饭边问他，跟谁出去了？去干什么了？</p> 
<p>这问话其实是有点风险的，花京院很有可能迎面甩他几句：管你什么事？告诉你你知道那是谁？你怎么还不去上班？</p> 
<p>好在没有，花京院坐在他对面，一边吃一边把同行的朋友姓名报上一遍，又说是一起去做一个小组作业，过几天大概就做好了，不用再出门。承太郎听着那几个名字，佐藤铃木之类，耳熟，但的确并不知道具体都是何许人也，于是得寸进尺地继续问这些都是谁啊？花京院不咸不淡地告诉他，是班长，是坐在我后座的同学，是国语课代表。</p> 
<p>他得了答案，点点头，继续吃饭，几口下肚，后知后觉地加一句：我就问问，你别嫌我烦。</p> 
<p>花京院头也不抬地嚼着嘴里的东西，嘟嘟囔囔地答道：我没觉得你烦。</p> 
<p>&nbsp;</p> 
<p>短休快要结束的时候，花京院也快要过生日了。暑假以来两人共处时气氛还算好，承太郎掂量着或许花京院的逆反期就快过去了，心里暗自先松了一口气。到了生日前一天，他问对方第二天要不要出去吃饭？花京院摇头说不要，就在家里吃，就在家里两个人过。</p> 
<p>他试探性地又问，要不要邀几个朋友之类的到家里来？我看你跟那几个同学关系蛮好的。花京院回头看着他，眉尾挑得很高，承太郎心想他又要生气了，但花京院说话的时候，语气并没有很激烈，花京院有些无奈地说：你是听不懂什么叫做就我们两个人过是不是啊。</p> 
<p>他点点头，说句，好吧，转身从对方的卧室退了出去。</p> 
<p>第二天清早他给花京院送了生日礼物，西阵织的领带和丝巾，外加一对祖母绿袖扣，对才上高一的花京院来说或许还太早了些，不怎么合适，但是他的确押对了宝，花京院很喜欢这样成熟的礼物，他喜欢被当做大人的感觉，当天心情也一直非常好，下午承太郎在厨房里烘樱桃派和准备晚餐的时候，他久违地也过来帮忙了。晚饭后承太郎把插着蜡烛的甜点推到他面前，花京院几乎是毫不犹豫地一鼓腮帮，将蜡烛吹灭了，监护人楞了一下，问道：许愿呢？忘记许愿了吧？</p> 
<p>“早就想好愿望了，”寿星一刀将樱桃派劈开，自信满满地答道，“这我怎么会忘。”</p> 
<p>气氛实在不赖，吃完了生日蛋糕，承太郎说要不要在家庭影院看部电影什么的，花京院也点头答应，两人于是一起移动到了沙发边上，他顺口问花京院，刚才许了什么愿望了？</p> 
<p>花京院沉默片刻，说，你闭上眼睛，我跟你说。</p> 
<p>这其中的逻辑漏洞其实很明显，听个愿望而已，又何必多此一举闭什么眼睛，但承太郎并没有怀疑，他坐在沙发上，坐在花京院身边，放松地闭上了眼睛。失去视觉之后依旧能感觉到什么东西正靠近了自己，虚浮地游离在他身旁十厘米左右的位置，承太郎微笑一下，心想或许是叛逆期的花京院今天难得地想给他一个拥抱，但是又碍于面子和自尊不敢立刻动手，没关系，他有的是耐心等他调整好心态再做这件事。</p> 
<p>承太郎闭着眼睛，最后等到的是嘴唇上一片滚烫而柔软的按压。</p> 
<p>他得到了一个青涩又坚定的吻。</p> 
<p>花京院吻了他，他心里一惊，当然条件反射地想要退开，然而没来得及行动，那虚浮地游离在体侧十厘米左右的触感却露出了真面目：法皇的触手立刻紧紧地贴上来缚住了他的行动，甚至有一条顺着脸颊划过来，遮住了他的眼睛。</p> 
<p>最强替身使者就这样被一个毫无战斗经验的少年给撂倒了，少年依旧停留在接吻的距离，说话间唇齿都因为紧张而打颤，每吐出一个音节，唇瓣几乎就要在承太郎的嘴唇上扫一下。</p> 
<p>“别动，承太郎，不准动。”</p> 
<p>“……”</p> 
<p>他没有回答，也没有动，花京院继续很紧张地说，如果你试图用白金之星把法皇扯开的话，我也会受伤的，这个你知道吧？</p> 
<p>承太郎缓缓地点点头。</p> 
<p>花京院犹豫了几秒，嘴唇再次贴了上来，他毫无经验，大概是凭着想象和道听途说来的知识，笨拙地试图取悦承太郎，湿滑灵活的舌头在承太郎的下唇上一遍遍匆匆忙忙地滑过，虽说技术不足，但吻得很投入，刚开始只是侧过头把脸贴上来，到后来伸出了手，柔软的指腹浸透了冷汗，凉丝丝地黏上承太郎的后颈，很用力地按着。</p> 
<p>隔着一片模模糊糊的绿色，承太郎依旧能感觉到对方的惊慌，仿佛闯了个无法自行解决收尾的大祸，还必须强装镇定，假装一切都在掌握之中。他轻轻叹了口气，稍微向前俯身，法皇没有阻止这样细小的动作变化，因此他得以更加贴近花京院，主动把自己的吻按上对方的嘴唇，给予安抚性的吮吸和摩擦，让颤颤巍巍的少年稍微有些自信。接吻这事他疏于练习，但好歹经验还在，天赋也不错，毕竟身体里流有部分意大利人的血，花京院则是白纸一张，自然没有什么高要求，他主动吻过去之后，法皇的触手很快一圈一圈地松散下来，直到他重获自由之身，得以腾出一只手托住花京院逐渐软倒下去的后背。</p> 
<p>最后一道触手被花京院收了回去，他恢复了光明，低头看着涨得通红的花京院。高中生抖得厉害，看着他的眼睛，眼角都潮了，对视了没几秒，脸一扬，作势又要亲过来，承太郎用手指顶住他的下巴，问道：</p> 
<p>“想清楚了？”</p> 
<p>“当，当然！”</p> 
<p>“想清楚了还结巴。”</p> 
<p>“不要开我玩笑！”花京院咬着牙，冷冰冰的手指在承太郎脖子后面无意识地绞紧了，“我是认真的！”</p> 
<p>亲吻紧接着又疾风骤雨地印到他的嘴唇上，力道很大，不得其法，承太郎心想自己这时候或许是不应该笑的，但花京院这个样子，他实在控制不住要弯弯嘴角。他伸出一只手探进花京院后脑的头发，稳住他，侧过头示范了一个不动用舌头但依旧足够亲怜密爱的吻，花京院绞在他脖子后面的手指随着这个吻渐渐地放松了下来，他搂住承太郎的脖子，鼻腔中漫出短促而黏腻的几下哼哼，在嘴唇分开的时候，立刻迫不及待地开口说道：</p> 
<p>“我爱你，承太郎。”</p> 
<p>主动吻了他的监护人托着他的后背，低声问他：</p> 
<p>“我比你大了快二十岁。我们都是男人。你是我养大的。这些你都知道吧？”</p> 
<p>“当然知道！”放在平时花京院应该会表现得深受冒犯，然而此时并不，黑暗中已出现了一线光明，他看到了希望，激动兴奋多过其他感情，“我才不在乎那个呢！”</p> 
<p>他当然不在乎，这个花京院是在他的羽翼下，在温室里长大的孩子，连摔跤都没摔过几次，然而承太郎在听到这句话的时候，突然发现其实他自己也并不在乎，年龄相差或是他人的眼光之类，远不及花京院也同样爱他这一点来得重要，而后者的真实性现在已经得到了肯定，其他的考虑突然都显得无聊且多余起来了。</p> 
<p>“我也爱你。”</p> 
<p>他心想，这最终还是要说出来的，有些话不能用一个吻或是一个眼神来替代。</p> 
<p>“而且和以前的那种……并不一样。”</p>
